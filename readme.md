# Boilerplate v1
A simple react starter project.

## Getting Started
Ensure that either Yarn or NPM are installed.

### Clone this repo
Begin by cloning this repo

### Delete the git info
```bash
rm -rf .git
```

### Start a new Git Repo
```bash
git init
```

### Set your git username
```bash
vi .git/config
```
### Add the following to your Git Config
```bash
[user]
    name = Your Name
    email = your@email.com
```

## Commands
### Install dependencies
```bash
yarn install
-or-
npm install
```
### Run dev-server
```bash
yarn run dev-server
-or-
npm run dev-server
```
